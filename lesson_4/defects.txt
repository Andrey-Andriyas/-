Утилитысapitalize - Принимает на вход текст, делает первую букву заглавной и возвращает этот же текстДефекты:- принимает пустую строку;
- принимает строку, состоящую из пробеловtrim - Принимает на вход текст и удаляет пробелы в начале, если они естьДефекты:- принимает пустую строкуto_list - Принимает на вход текст с разделителем и возвращает список строкДефекты:- при установке разделителя «:» выводится список с разделителем «,»contains - Возвращает `True`, если строка содержит искомый символ и `False` - если нетДефекты:- принимает пустую строку;
- принимает строку, состоящую из пробеловdelete_symbol - Удаляет все подстроки из переданной строкиДефекты:- принимает пустую строку;
- принимает строку, состоящую из пробеловstarts_with - Возвращает `True`, если строка начинается с заданного символа и `False` - если нетДефекты:- принимает пустую строку;
- принимает строку, состоящую из пробеловend_with - Возвращает `True`, если строка заканчивается заданным символом и `False` - если нетДефекты:- принимает пустую строку;
- принимает строку, состоящую из пробеловis_empty - Возвращает `True`, если строка пустая и `False` - если нетlist_to_string - Преобразует список элементов в строку с указанным разделителемДефекты:- принимает пустую строку
